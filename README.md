# 4.-Employee-Salary-Viewer
Objective:
To retrieve and display the salary of an employee using their employee ID through a JDBC program.
Description:
The user enters an employee ID, and the system retrieves the employee's salary from the database. Demonstrates SELECT queries, ResultSet processing, and Java console interaction.
Requirements:
Java:
Basic console input
JDBC connection
ResultSet reading
JDBC Concepts:
Parameterized SELECT
ResultSet getInt(), getDouble(), getString()
Handling "Record not found" cases
SQL Table:
employees(emp_id INT PRIMARY KEY, name VARCHAR(50), salary DOUBLE)
